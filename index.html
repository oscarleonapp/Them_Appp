Comandos GIT

Abrir enlace en una nueva pestaña

Para realizar esta acción bastará con añadir el atributo target a la etiqueta, vea el siguiente ejemplo:

<a href="#" target="_blank"> Enlace </a>

Ejemplo: Click

NOTA: Por seguridad se debe agregar el atributo rel="noopener noreferrer"

Cambiar el título de la página

Dirigirse al código y ubicar la etiqueta <head></head> dentro de ella agregar la etiqueta <title></title> y ahora escribe el titulo que desees.

Este aparecerá en la parte superior de tu navegador, el que indica el nombre de tu pestaña.

<!DOCTYPE html>
<html>
<head>
<title>Mi página web</title>
</head>
<body>
</body>
</html>

Enlace con imágenes

Puedes lograr esto rodeando la itequeta <img src=""> con la etiqueta <a>

<a href="www.google.com"><img source="/gatos.png" alt="gatos"></a>

Fuente: National Geographic en español

Entidades de caracteres

En HTML existen caracteres que no pueden ser utilizados ya que son interpretados por el navegador como código.

Caracter Entidad Nota
& &amp; Interpretado como el comienzo de una entidad HTML
< &lt; Interpretado como la apertura de una etiqueta
> &gt; Interpretado como el cierre de una etiqueta
" &quot; Intepretado como apertura o cierre del valor de un atributo
