Comandos GIT

Abrir enlace en una nueva pestaña

Para realizar esta acción bastará con añadir el atributo target a la etiqueta <a href="#" target="_blank"> Enlace </a>

Ejemplo: Click

NOTA: Por seguridad se debe agregar el atributo rel="noopener noreferrer"

Cambiar el título de la página

Dirigirse al código y ubicar la etiqueta <head></head> dentro de ella agregar la etiqueta <title></title> y ahora escribe el titulo que desees.

Este aparecerá en la parte superior de tu navegador, el que indica el nombre de tu pestaña.

Enlace con imágenes

Puedes lograr esto rodeando la itequeta <img> con la etiqueta <a>

Ejemplo: a href="www.google.com"> img source="/gatos.png" alt="gatos"> /a>
